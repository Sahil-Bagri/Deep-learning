# 📌 Topic Modeling using LDA, LSA, and NMF

# 🚀 Project Overview

# This project demonstrates unsupervised topic modeling using three popular techniques:

# Latent Dirichlet Allocation (LDA)
# Latent Semantic Analysis (LSA)
# Non-negative Matrix Factorization (NMF)

# The goal is to extract meaningful topics from a collection of text documents and compare how each model performs in identifying hidden patterns.

# Topic modeling helps represent documents as a mixture of topics, where each topic is a distribution of words .

# 🧠 Key Concepts Used
# 🔹 LDA (Latent Dirichlet Allocation)
# Probabilistic model
# Assumes documents are a mixture of topics
# Each topic is a distribution over words
# 🔹 LSA (Latent Semantic Analysis)
# Based on Singular Value Decomposition (SVD)
# Captures relationships between words and documents
# Useful for dimensionality reduction
# 🔹 NMF (Non-negative Matrix Factorization)
# Matrix decomposition technique
# Produces interpretable topics
# Works well with TF-IDF features
